# 🔐 Password Manager (Python)

A simple command-line password manager built with Python. This project allows users to store and view account credentials using text files.

## Features

- Add account credentials
- View saved credentials
- Store data in text files
- Simple command-line interface

## Technologies Used

- Python 
- File Handling
- Functions

## Project Structure

```
password-manager-python/
│── password_manager.py
│── sample_password.txt
│── sample_master.txt
│── .gitignore
│── README.md
```

## How to Run

```bash
git clone https://github.com/aaravpandey198/password-manager-python.git
cd password-manager-python
python password_manager.py
```
